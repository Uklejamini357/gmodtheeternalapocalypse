This gamemode requires three things:

1. Your map must have a navmesh as the zombies are nextbots (google gmod navmesh if you dont know what this is or how to make one)
(or just do nav_generate, if it can't generate, simply enable sv_cheats, enable nav_edit then aim at where to start the navmesh generation
and use nav_mark_walkable command, then do nav_generate [you can also use Navmesh tool])
--Edit: There's also Navmesh Optimizer which generates navmesh nearly 100x faster.

2. Your server must have counter strike source mounted. Otherwise, some props, boss and faction loot caches
will be errors and you won't be able to hear some sounds since some sounds are from CS:S.
--Edit: Gmod already got CSS content packed into the game, so yeah.

3. You need to create a config file for your map.

You'd need zombies, loots, airdrops, trader spawnpoints.
Optionally you can also put player and taskdealer spawnpoints, but that's up to you.

By opening F2, you can go into admin commands and toggle Admin Mode.
Upon doing so, you'll respawn in the same spot with an admin tool equipped.
And now to open what you want to spawn in, select what you need to use.
Go to Spawns category, and pick what you need to use.
You can also use Admin Eyes to track where you spawned the stuff for specific type.

If you manage to screw up one of the spawns, no worries. Simply go to admin menu, map management and delete one of the id's from selected categories. 
Or you can cleanup all and begin from the start again. Your choice.


That's all!
If you have all of these above set up, you will have a niche apocalyptic experience.

Optional things to set up:

- Openworld transitions:
Allows smooth transitions between maps, useful if you are planning a nice.
Recommendation: Around 3-4 areas with different danger levels for openworld exploration would suffice

# Adding custom content:
Adding your own items and weapons is as easy as it gets. You just need to do the following:

1. Add the items in one of the files from "items" folder. You can even create a new file without modifying other files.
Items folder is where all the items are loaded from.

Other steps are optional. If you are considering to add other ways to obtain the item, you might want to add the item to the loot table. 

2. Add the item to the loot table in:
sh_loot.lua - items included in the loot tables drop from loot caches, airdrops, faction loot caches and bosses.
Overall this determines what can drop from the loots 

sh_spawnables.lua - controls which props and structures are available for players to build

sh_config.lua - configurate max zombies, max loot and a lot more

For other things, you may consider learning how the gamemode works.
For example


You can also edit the entire gamemode if you are making your own version. You just need to know gmod lua, though.

Other things I also recommend (not required):
- Installing Visual Studio Code (A text editor app for coding)
- Initializing local Git repository (That way you can a local repository with your changes and versions.)
