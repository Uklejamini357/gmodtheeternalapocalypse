The Eternal Apocalypse is a revamp of an old apocalyptic gamemode After The End, originally made by a legend LegendofRobbo back in 2015.

The original gamemode is no longer maintained, so I thought to revive it with more fun and mechanics.

Comes with many more features, including:
- Prestiging,
- Numerous bug fixes found in the original,
- Stats menu,
- Translation support,
- Mastery,
- Quests,
- Dynamic difficulty scaling,
- and many more!

### For which game is this for?
Garry's Mod.
